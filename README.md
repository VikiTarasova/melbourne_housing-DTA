# melbourne_housing-DTA
# Melbourne Housing Snapshot Dataset

## Про датасет

### Melbourne Housing Snapshot

## Огляд

Датасет Melbourne Housing Snapshot містить інформацію про продаж житлової нерухомості в Мельбурні, Австралія.

Він включає характеристики об’єктів нерухомості, інформацію про місцезнаходження та дані про продажі. Датасет можна використовувати для аналізу ринку нерухомості та задач машинного навчання, таких як прогнозування цін на житло.

Цільова змінна — **Price** (ціна продажу нерухомості).

## Ознаки датасету

Датасет містить:

- числові ознаки: Rooms, Price, Distance, Landsize, BuildingArea, YearBuilt, координати;
- категоріальні ознаки: Suburb, Type, Regionname, Method;
- часові дані: дата продажу нерухомості;
- цільова змінна: Price.
